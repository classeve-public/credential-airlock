# Credential Airlock

Credential Airlock is a Classeve open-source security product.

Copyright (c) 2026 Classeve.

Created by Ranjit Barnala and released by Classeve under the Apache License,
Version 2.0. See [LICENSE](LICENSE) for the full license text.

This product is self-hosted. Classeve does not receive, store, proxy, analyze,
or phone home with user credentials. Credential Airlock is designed so the local
operator keeps control of the vault, policy, audit log, and release artifact
they choose to run.
